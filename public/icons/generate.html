ClubIt App Icon Generator

How to use:
  1. Open this file in a browser: http://localhost:3000/icons/generate.html
  2. Click the download buttons below to save each PNG size.
  3. Place the downloaded files in public/icons/ alongside this file.
  4. The PWA manifest already references /icons/icon-192x192.png and /icons/icon-512x512.png.
